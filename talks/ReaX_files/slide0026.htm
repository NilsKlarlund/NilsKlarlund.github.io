The reaction tree
nThe root is an empty node (document node)
nIt holds the external event or injected event
nAfter an event is handled,
nthe signaled events are appended as children in left-to-right order with time
neach child is accompanied by its pre-events, which are siblings inserted in front of it
nWhen an event has executed
nits post-events are inserted after it as siblings, and
nthe next node in pre-order traversal is selected for handling
